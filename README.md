## Development Setup

### Prerequisites
- Node.js
- npm

### Installation

```
# Install dependencies
npm install
```

### Development Commands

```
# Build the extension
npm run build
```

### Loading in Chrome

1. Open `chrome://extensions` in Chrome
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `dist` directory
